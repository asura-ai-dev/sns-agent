## Failed Tests / Known Issues

- apps/web の `next build` が Node.js v25.8.1 環境でハングする（SWC コンパイラが子プロセス IPC で無応答になる）。TypeScript 型チェック（tsc --noEmit）は正常に通る。Node.js v25 は Next.js の公式サポート範囲外の可能性があり、Node.js 20 LTS または 22 LTS 環境での動作確認が必要
- `pnpm build` は web を含めると上記のハングにより完了しない。`pnpm turbo run build --filter='!@sns-agent/web'` で 12/12 パッケージの build は成功する

## Key Decisions

- ESLint は flat config 形式（eslint.config.mjs）を採用（eslint v9 対応）
- apps/web の tsconfig.json はルートの extends を使わず独立定義（Next.js の rootDir/outDir 制約との競合回避）
- pnpm.onlyBuiltDependencies で esbuild, sharp のビルドスクリプトを許可
- Node.js v25 + Next.js 15 の互換性問題はこのチケットの scope 外として known_gaps に記録
