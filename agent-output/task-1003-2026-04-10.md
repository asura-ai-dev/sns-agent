# Handoff: Task 1003 - packages/core ドメインモデルとインターフェース定義

## Spec Alignment

spec.md の Phase 1 基盤構築における packages/core の定義に対応。RBAC (AC-21)、投稿状態遷移 (AC-3/AC-6)、Provider 抽象 (非機能要件: 拡張性)、暗号化 (非機能要件: セキュリティ) の型基盤を提供する。

## Phase

Implementation (Phase 1)

## completed

- ドメインエンティティ型 14 種を `packages/core/src/domain/entities.ts` に定義 (Workspace, User, AgentIdentity, SocialAccount, Post, ScheduledJob, ConversationThread, Message, UsageRecord, BudgetPolicy, LlmRoute, SkillPackage, ApprovalRequest, AuditLog)
- Repository インターフェース 7 種を `packages/core/src/interfaces/repositories.ts` に定義 (AccountRepository, PostRepository, ScheduledJobRepository, UsageRepository, BudgetPolicyRepository, LlmRouteRepository, AuditLogRepository)
- SocialProvider インターフェースと入出力型を `packages/core/src/interfaces/social-provider.ts` に定義
- JobQueue インターフェースを `packages/core/src/interfaces/job-queue.ts` に定義
- ドメインエラー 7 種を `packages/core/src/errors/domain-error.ts` に定義 (DomainError, ValidationError, AuthorizationError, NotFoundError, BudgetExceededError, ProviderError, RateLimitError)
- RBAC ポリシー (Permission 型, rolePermissions マッピング, checkPermission ヘルパー) を `packages/core/src/policies/rbac.ts` に定義
- 暗号化ユーティリティ (AES-256-GCM) を `packages/core/src/domain/crypto.ts` に定義
- `@sns-agent/config` を workspace:\* で依存追加し、Platform, Role 等を import
- `packages/core/src/index.ts` から全エクスポートを接続
- `tsc` ビルド成功
