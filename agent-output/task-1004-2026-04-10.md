## 重要判断

- SQLite の timestamp は integer (Unix timestamp) モードを使用。Drizzle の `{ mode: "timestamp" }` で Date オブジェクトとの変換を自動化
- JSON カラムは `text({ mode: "json" })` を使用（SQLite には native JSON 型がないため）
- decimal カラム（usage_records.estimated_cost_usd, budget_policies.limit_amount_usd）は `real` 型を使用（SQLite には decimal 型がないため）
- better-sqlite3 のネイティブビルドを pnpm の onlyBuiltDependencies に追加が必要
