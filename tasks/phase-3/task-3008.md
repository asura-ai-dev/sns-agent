# Task 3008: Web UI - 設定画面（アカウント接続・ユーザー管理）

## 目的

ワークスペース設定画面を実装する。SNS アカウント接続管理、ユーザー管理（RBAC）の UI を提供する。LLM ルーティングと予算設定は Phase 4/5 で追加する。

- `apps/web/src/app/(dashboard)/settings/accounts/page.tsx` - アカウント接続管理
- `apps/web/src/app/(dashboard)/settings/users/page.tsx` - ユーザー管理
- `apps/web/src/components/settings/` - 設定関連コンポーネント

## 実装指示

1. **設定トップページ** `settings/page.tsx`
   - タブまたはサイドバーで設定カテゴリを選択する構成
   - カテゴリ: アカウント接続、ユーザー管理、（LLM ルーティング - 後続）、（予算 - 後続）
   - デフォルトは「アカウント接続」

2. **アカウント接続管理** `settings/accounts/page.tsx`
   - 接続済みアカウント一覧
     - 各行: プラットフォームアイコン、アカウント名、ステータスバッジ（active/expired/error）、トークン有効期限、アクション
     - ステータスが expired の場合は警告表示 + 「再接続」ボタン
   - 「新規接続」ボタン
     - プラットフォーム選択（X, LINE, Instagram のカード）
     - 選択後に OAuth フローを開始（新しいウィンドウで OAuth URL を開く）
   - アカウント切断ボタン（確認ダイアログ付き）
   - ロールに応じた表示制御
     - viewer/operator/editor: 一覧表示のみ、接続/切断ボタンは非表示
     - admin/owner: 全操作可能

3. **ユーザー管理** `settings/users/page.tsx`
   - ユーザー一覧テーブル: 名前、メール、ロール、作成日
   - ロール変更: ドロップダウンで変更（admin/owner のみ）
   - ユーザー招待（メールアドレス + ロール指定）
   - エージェント ID 一覧: 名前、ロール、API キー（マスク表示）、作成日
   - エージェント ID 作成ボタン

4. **ロール表示**
   - 各ロールに色付きバッジ: viewer(灰), operator(青), editor(緑), admin(橙), owner(赤), agent(紫)
