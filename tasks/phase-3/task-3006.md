# Task 3006: Web UI - 投稿一覧・投稿作成画面

## 目的

投稿一覧画面（フィルタ、ページネーション）と投稿作成画面（SNS 選択、コンテンツ入力、バリデーション、プレビュー、送信）を実装する。

## scope

- `apps/web/src/app/(dashboard)/posts/page.tsx` - 投稿一覧ページ
- `apps/web/src/app/(dashboard)/posts/new/page.tsx` - 投稿作成ページ
- `apps/web/src/components/posts/` - 投稿関連コンポーネント

## 実装指示

1. **投稿一覧ページ** `posts/page.tsx`
   - フィルタバー: プラットフォーム選択（チェックボックス or タブ）、ステータスフィルタ、日付範囲
   - 投稿テーブル/カードリスト
     - 各行: SNS アイコン、アカウント名、テキスト冒頭、ステータスバッジ、作成日時、アクション（編集、削除、公開）
     - ステータスバッジ: draft(灰), scheduled(青), published(緑), failed(赤), deleted(灰取消線)
   - ページネーション（DaisyUI の join + btn）
   - 「新規投稿」ボタン -> `/posts/new` へ遷移
   - モバイルではカードレイアウトに切り替え

2. **投稿作成ページ** `posts/new/page.tsx`
   - ステップ形式またはシングルページフォーム
   - **SNS 選択**: 接続済みアカウントをドロップダウンで選択。プラットフォームアイコン付き
   - **コンテンツ入力**:
     - テキストエリア（文字数カウンター付き。プラットフォーム別上限を表示）
     - メディアアップロード（画像/動画。ドラッグ&ドロップ対応）。v1 はプレビューのみでアップロード先はプレースホルダ可
   - **バリデーション表示**: リアルタイムで文字数超過等を警告表示
   - **プレビュー**: 投稿内容のプレビュー表示（SNS ごとの見た目を模倣する必要はないが、テキスト + メディアの確認ができること）
   - **送信ボタン**: 「下書き保存」と「即時投稿」の 2 つのアクション
   - 送信後は投稿詳細ページまたは一覧にリダイレクト

3. **コンポーネント分割**
   - `PostList.tsx`: 投稿一覧テーブル/カード
   - `PostFilters.tsx`: フィルタバー
   - `PostForm.tsx`: 投稿作成フォーム
   - `PostPreview.tsx`: プレビュー表示
   - `CharacterCounter.tsx`: 文字数カウンター（プラットフォーム別上限対応）
   - `StatusBadge.tsx`: ステータスバッジ

4. **API 連携**
   - 投稿一覧: Server Component で API から取得
   - 投稿作成: Client Component で form submit -> server action or API call
