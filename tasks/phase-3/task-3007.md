# Task 3007: Web UI - 予約カレンダー画面

## 目的

予約投稿をカレンダー形式で表示・管理する画面を実装する。月/週ビュー、予約の詳細表示、カレンダーからの予約作成導線を提供する。

## 実装指示

1. **カレンダーライブラリ**
   - 軽量なカレンダーライブラリを使用する（推奨: `@fullcalendar/react` または自前の月/週グリッド実装）
   - DaisyUI のスタイルと統合する
   - 重量級ライブラリの導入が困難な場合、最低限月ビューのグリッド表示を自前で実装する

2. **カレンダーページ** `calendar/page.tsx`
   - ビュー切り替え: 月 / 週（タブまたはトグル）
   - 月ビュー: 日付グリッドに予約投稿をドット/小カードで表示
   - 週ビュー: 7 日間のタイムライン表示
   - 各予約をクリックすると詳細モーダルが開く
   - 「新規予約」ボタンまたは日付クリックで予約作成導線へ

3. **予約詳細モーダル** `components/calendar/ScheduleDetailModal.tsx`
   - DaisyUI の modal コンポーネント
   - 予約情報: 投稿テキスト冒頭、SNS、アカウント、予約日時、ステータス
   - アクション: 日時変更、キャンセル、投稿詳細へのリンク

4. **月ナビゲーション**
   - 前月/次月、今日ボタン
   - 表示中の年月を表示

5. **ステータス色分け**
   - pending: primary カラー
   - succeeded: 緑
   - failed: 赤
   - retrying: warning カラー

6. **API 連携**
   - `GET /api/schedules` から予約一覧を取得する
   - 日付範囲パラメータでカレンダー表示範囲のデータのみ取得する
