# Task 3002 done_when

- `sns post create --platform x --account main --text "Hello" --json` が投稿を作成し JSON で結果を返す
- `sns post create --platform x --account main --file draft.md` がファイルから本文を読み込んで投稿を作成する
- `sns post list --platform x --json` が X の投稿一覧を JSON で出力する
- `sns post publish <id>` が下書きを即時公開する
- `sns post delete <id>` が投稿を削除する
- `sns schedule create --post <id> --at "2026-04-12T09:00:00+09:00"` が予約を作成する
- `sns schedule list --json` が予約一覧を JSON で出力する
- `sns schedule cancel <id>` が予約をキャンセルする
- `sns inbox list --platform x --json` がスレッド一覧を出力する（API 未実装の場合は適切なエラーメッセージ）
- 全コマンドで必須パラメータ未指定時にヘルプが表示される
- 全コマンドが終了コード 0（成功）/ 1（エラー）を返す
