# Task 6001 done_when

- 書き込み系 API（投稿作成、アカウント接続等）を実行すると audit_logs テーブルにレコードが記録される
- 記録されるログに actor_id, actor_type, action, resource_type, resource_id, platform, request_id が含まれている
- AuditLogRepository に update / delete メソッドが存在しないこと
- `GET /api/audit` で監査ログ一覧が取得できる
- actorId, action, platform, 日付範囲でフィルタできる
- admin 未満のロールが `GET /api/audit` を呼ぶと 403 が返る
- `/settings/audit` に監査ログ画面が表示される
- フィルタバーでフィルタ条件を指定して絞り込みができる
- 行クリックで詳細モーダルが開き、input_summary と result_summary が表示される
- 機密情報（トークン、API キー）が input_summary に含まれていないこと
