# Task 6002 done_when

- agent ロールが投稿を公開しようとすると承認リクエストが作成され、投稿は公開されない
- admin ロールが投稿を公開する場合は承認不要で即時公開される
- `GET /api/approvals?status=pending` で承認待ちリストが取得できる
- `POST /api/approvals/:id/approve` で承認が完了し、元の投稿が公開される
- `POST /api/approvals/:id/reject` で却下され、投稿は draft に戻る
- 承認・却下が監査ログに記録される
- 予算超過（require-approval）時に承認リクエストが作成される
- Header の通知ベルに承認待ち件数バッジが表示される
- 24 時間経過した pending リクエストが expired に更新される
- admin 未満のロールが承認/却下操作を試みると 403 が返る
