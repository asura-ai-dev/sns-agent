# Task 6003 done_when

- Webhook で受信したメッセージが conversation_threads と messages テーブルに保存される
- `GET /api/inbox` でスレッド一覧が取得でき、platform フィルタが動作する
- `GET /api/inbox/:threadId` でスレッド内メッセージ一覧が取得できる
- `POST /api/inbox/:threadId/reply` で返信が送信され、Provider の sendReply が呼ばれる
- 返信が messages テーブルに outbound として記録される
- `/inbox` にアクセスすると 2 カラムの受信トレイ画面が表示される
- スレッド一覧に SNS アイコン、相手名、最新メッセージ冒頭が表示される
- スレッドを選択するとメッセージ一覧が表示される（inbound 左、outbound 右）
- 返信テキストを入力して送信できる
- モバイルで左右パネルがスライド切り替えになる
