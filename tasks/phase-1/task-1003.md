- `packages/core/src/interfaces/` - Repository, SocialProvider, JobQueue インターフェース

## 実装指示

1. **ドメインエンティティ型**を `domain/` に定義する（TypeScript type/interface）
   - `Workspace`, `User`, `AgentIdentity`
   - `SocialAccount` (platform, status, capabilities, token_expires_at 等)
   - `Post` (status 遷移: draft -> scheduled -> publishing -> published / failed / deleted)
   - `ScheduledJob` (status 遷移: pending -> locked -> running -> succeeded / failed / retrying)
   - `ConversationThread`, `Message`
   - `UsageRecord`, `BudgetPolicy`
   - `LlmRoute`, `SkillPackage`
   - `ApprovalRequest`, `AuditLog`

2. **Repository インターフェース**を `interfaces/` に定義する
   - `AccountRepository`: findById, findByWorkspace, create, update, delete
   - `PostRepository`: findById, findByWorkspace, create, update, delete, findByIdempotencyKey
   - `ScheduledJobRepository`: findById, findPendingJobs, create, update, lockJob
   - `UsageRepository`: record, aggregate
   - `BudgetPolicyRepository`: findByWorkspace, create, update, delete
   - `LlmRouteRepository`: findByWorkspace, resolve, create, update, delete
   - `AuditLogRepository`: create, findByWorkspace（UPDATE/DELETE メソッドなし）
   - 各インターフェースは具体的な DB 実装に依存しないこと

3. **SocialProvider インターフェース**を `interfaces/social-provider.ts` に定義する
   - design.md セクション 6 の定義に準拠する
   - 入出力型（ConnectAccountInput, PublishPostInput, ValidationResult 等）も同ファイルに定義

4. **JobQueue インターフェース**を `interfaces/job-queue.ts` に定義する
   - `enqueue(job)`, `dequeue()`, `complete(jobId)`, `fail(jobId, error)`, `retry(jobId)`

5. **ドメインエラー**を `errors/` に定義する
   - `DomainError` (base), `ValidationError`, `AuthorizationError`, `NotFoundError`
   - `BudgetExceededError`, `ProviderError`, `RateLimitError`

6. **RBAC ポリシー**を `policies/rbac.ts` に定義する
   - `Permission` 型: 操作の列挙（'account:read', 'account:connect', 'post:read', 'post:create', 'post:publish' 等）
   - `rolePermissions`: Role -> Permission[] のマッピング
   - `checkPermission(role, permission)`: boolean を返すヘルパー関数

7. **暗号化ユーティリティ**を `domain/crypto.ts` に定義する
   - `encrypt(plaintext, key)`, `decrypt(ciphertext, key)` - AES-256-GCM
