# Task 2004 done_when

- `POST /api/posts` に `{ socialAccountId, contentText, publishNow: false }` を送ると下書き投稿が作成され、status が "draft" で返る
- `POST /api/posts` に `publishNow: true` を送ると即時公開が試みられ、成功時に status が "published" で返る
- テキストが X の 280 文字制限を超える場合、`POST /api/posts` が ValidationError（400）を返す
- `PATCH /api/posts/:id` で下書きのテキストを更新できる。公開済み投稿の更新は 400 になる
- `POST /api/posts/:id/publish` で下書きを公開でき、platform_post_id が記録される
- `DELETE /api/posts/:id` で投稿が論理削除される（status が "deleted" に変わる）
- `GET /api/posts` にフィルタ（platform, status）を指定して一覧取得できる
- `X-Idempotency-Key` ヘッダ付きで同じ POST を 2 回送った場合、2 回目は既存投稿を返し、重複作成されない
- viewer ロールが投稿一覧を取得でき、作成・公開操作は RBAC で制御される
