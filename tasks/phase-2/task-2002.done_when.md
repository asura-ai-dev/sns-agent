# Task 2002 done_when

- `GET /api/accounts` が接続済みアカウント一覧を返す（200、`{ data: [...] }` 形式）
- `POST /api/accounts` に `{ platform: "x" }` を送ると OAuth URL を含むレスポンスが返る
- `GET /api/accounts/:id` がアカウント詳細を返す（トークン本体は含まない）
- `DELETE /api/accounts/:id` がアカウントの status を 'revoked' に更新する
- OAuth トークンが暗号化されて `credentials_encrypted` カラムに保存されている（平文でない）
- `token_expires_at` がレスポンスに含まれ、期限切れ間近の場合に warning フラグが付く
- viewer ロールがアカウント一覧を取得でき、接続/切断操作は 403 になる
- admin ロールが接続/切断操作を実行できる
- `packages/core/src/usecases/account.ts` に上記ユースケースが実装されている
