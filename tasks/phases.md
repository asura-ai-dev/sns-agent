# SNS Agent v1 - フェーズ定義

## フェーズ一覧

| Phase | 名称                   | チケット範囲 | 依存             | 概要                                                                 |
| ----- | ---------------------- | ------------ | ---------------- | -------------------------------------------------------------------- |
| 1     | 基盤構築               | 1001-1006    | なし             | モノレポ骨格、共通設定、DB、core ドメイン、API 骨格、SDK             |
| 2     | 認証・アカウント・投稿 | 2001-2006    | Phase 1          | RBAC、SocialAccount、Provider 抽象、X プロバイダ、投稿管理、予約投稿 |
| 3     | CLI + Web UI 基本      | 3001-3008    | Phase 2          | CLI コマンド群、Web UI レイアウト・各画面                            |
| 4     | 全 SNS 対応 + 使用量   | 4001-4005    | Phase 2          | LINE/Instagram プロバイダ、使用量記録、予算ポリシー、コスト可視化    |
| 5     | AI 連携                | 5001-5005    | Phase 3, Phase 4 | LLM アダプタ、ルーティング、Agent Gateway、skills、チャット UI       |
| 6     | 監査・承認・受信       | 6001-6004    | Phase 2          | 監査ログ、承認フロー、受信・会話管理、統合テスト                     |

## 依存関係図

```
Phase 1 (基盤)
  |
  v
Phase 2 (認証・アカウント・投稿)
  |         |            |
  v         v            v
Phase 3   Phase 4      Phase 6
(CLI/UI)  (全SNS/使用量) (監査・承認・受信)
  |         |
  v         v
Phase 5 (AI連携)
        |
        v
  Task 6004 (統合テスト - 全フェーズ完了後)
```

## 並列実行可能なフェーズ

- Phase 3 と Phase 4 は Phase 2 完了後に並列実行可能
- Phase 6 の task-6001 から task-6003 は Phase 2 完了後に着手可能（Phase 3-5 と並列可能）
- Phase 5 は Phase 3 と Phase 4 の両方が完了してから着手
- Task 6004（統合テスト）は全フェーズ完了後に実行
