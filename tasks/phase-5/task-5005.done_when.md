# Task 5005 done_when

- `/settings/llm` にアクセスすると LLM ルーティング設定画面が表示される
- ルート一覧に既存のルートが Platform アイコン付きで表示される
- ルート作成フォームから Provider, Model, Platform, Action を指定してルートを作成できる
- ルートの編集・削除が可能である
- `/skills` にアクセスすると Skills 管理画面が表示される
- パッケージ生成ボタンから Platform と LLM Provider を選択してパッケージを生成できる
- 生成完了後に一覧にパッケージが追加される
- 有効/無効トグルで DaisyUI の toggle が動作し、API に反映される
- パッケージ詳細にアクション一覧とパラメータが表示される
- ルート 0 件、パッケージ 0 件の場合に適切な空状態メッセージが表示される
