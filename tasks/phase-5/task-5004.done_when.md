# Task 5004 done_when

- `/agents` にアクセスするとチャット画面が表示される
- テキストを入力して送信すると AI からの応答がリアルタイムで表示される（SSE ストリーミング）
- ストリーミング中にタイピングインジケーターが表示される
- LLM が skill アクションを提案した場合、ActionPreview カードにアクション名・パラメータが表示される
- 「実行」ボタンを押すとアクションが実行され、結果がチャットに表示される
- 「キャンセル」ボタンを押すとプレビューが閉じる
- 左パネルに会話履歴が表示され、過去の会話を選択できる
- 新規会話作成ボタンが動作する
- モバイルでパネルが切り替え表示になる
- ストリーミング中は送信ボタンが無効化される
