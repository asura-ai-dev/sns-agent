# Task 5004: Web UI - チャットインターフェース

## 目的

Web UI にチャット形式の AI オペレータ画面を実装する。メッセージ送受信、SSE ストリーミング表示、skill 実行のプレビュー・承認 UI を構築する。

## 実装指示

1. **チャットページ** `agents/page.tsx`
   - 左パネル: 会話履歴リスト（過去の会話を選択して表示）
   - 右パネル: チャットエリア（メッセージ表示 + 入力）
   - モバイルではパネル切り替え

2. **チャットコンポーネント** `components/chat/`
   - `ChatContainer.tsx`: メッセージリスト + 入力エリアのコンテナ
   - `MessageBubble.tsx`: メッセージバブル
     - ユーザーメッセージ: 右寄せ、primary カラー
     - AI メッセージ: 左寄せ、neutral カラー
     - ストリーミング中: タイピングインジケーター表示
   - `ChatInput.tsx`: テキスト入力 + 送信ボタン。Enter で送信、Shift+Enter で改行
   - `ActionPreview.tsx`: skill 実行プレビューカード
     - アクション名、対象 SNS、パラメータのプレビュー
     - 「実行」「キャンセル」ボタン
     - 実行中はローディング、実行後は結果表示
   - `ConversationList.tsx`: 左パネルの会話履歴

3. **SSE ストリーミング**
   - `POST /api/agent/chat` の SSE レスポンスを受信し、リアルタイムでメッセージバブルに表示する
   - EventSource または fetch + ReadableStream で受信する
   - ストリーミング中は送信ボタンを無効化する

4. **承認フロー UI**
   - LLM が skill アクションを提案した場合:
     1. ActionPreview カードでプレビューを表示
     2. ユーザーが「実行」を押すと `POST /api/agent/execute` を呼ぶ
     3. 実行結果をチャットタイムラインに追加する
   - ユーザーが「キャンセル」を押すとプレビューを閉じてチャットを続行する

5. **会話管理**
   - 新規会話作成ボタン
   - 会話ごとにメッセージ履歴を保持する
   - API の `GET /api/agent/history` から履歴を取得する
