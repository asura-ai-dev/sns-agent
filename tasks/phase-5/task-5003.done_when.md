# Task 5003 done_when

- `POST /api/skills/generate` に `{ platform: "x", llmProvider: "openai" }` を送ると skill パッケージが生成される
- 生成された manifest に platform, provider, actions 配列が含まれている
- actions にはプラットフォームの capability に応じたアクション（create_post, list_posts 等）が含まれる
- 各アクションに parameters（JSON Schema 形式）と permissions が定義されている
- `PATCH /api/skills/:id` で `{ enabled: true }` を送るとパッケージが有効化される
- `GET /api/skills/:id/manifest` で manifest JSON を取得できる
- `GET /api/skills` でパッケージ一覧が取得できる
- 有効化された skill が Agent Gateway の system prompt に反映される（getActiveSkills が正しく動作する）
- `pnpm build --filter @sns-agent/skills` がエラーなく完了する
