# Task 5002 done_when

- `POST /api/agent/chat` に `{ message: "Xに投稿して" }` を送ると LLM レスポンスが返る
- LLM が skill 実行を要求した場合、dry-run プレビュー（何が実行されるかの説明）が JSON で返る
- `POST /api/agent/execute` で承認済みアクションが実行され、core のユースケースに委譲される
- skill アクションの引数が JSON Schema に違反する場合にバリデーションエラーが返る
- actor の権限が不足している場合に 403 が返る
- 予算超過時に BudgetExceededError が返る
- `GET /api/agent/history` でチャット履歴が取得できる
- `packages/skills/src/manifest/types.ts` に SkillManifest, SkillAction 型が定義されている
- `packages/skills/src/runtime/executor.ts` に validateSkillAction, executeSkillAction, dryRunSkillAction が実装されている
- デフォルトの実行モードが approval-required になっている
