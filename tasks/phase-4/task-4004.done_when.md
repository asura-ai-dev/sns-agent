# Task 4004 done_when

- `POST /api/budget/policies` でポリシーを作成できる（scope_type, period, limit_amount_usd, action_on_exceed を指定）
- `GET /api/budget/policies` でポリシー一覧を取得できる
- `GET /api/budget/status` で各ポリシーの現在消費額・上限・消費率を取得できる
- 予算上限を超えた状態で投稿公開を試みた場合、action が `block` なら BudgetExceededError（403）が返る
- action が `warn` の場合は公開が続行され、レスポンスに warning が含まれる
- 消費率が 80% に達した場合に warn が返る
- ポリシーの scope 優先度（endpoint > platform > workspace）が正しく評価される
- admin 未満のロールがポリシー作成/更新/削除を試みると 403 が返る
