# Task 4005: Web UI - 使用量・予算画面

## 目的

API 使用量と推定コストをグラフ表示し、予算ポリシーの設定・管理を行う Web UI 画面を実装する。

## depends_on

[3004, 4003, 4004]

- agent-docs/spec.md - AC-14

## scope

- `apps/web/src/app/(dashboard)/usage/page.tsx` - 使用量ページ
- `apps/web/src/components/usage/` - 使用量関連コンポーネント
- `apps/web/src/app/(dashboard)/settings/budget/page.tsx` - 予算設定ページ

## 実装指示

1. **使用量ページ** `usage/page.tsx`
   - 期間切り替え: 日次 / 週次 / 月次（タブ）
   - プラットフォームフィルタ: 全体 / X / LINE / Instagram / LLM
   - **サマリカード**: 合計コスト、リクエスト数、成功率、前期比
   - **グラフ**: 使用量の推移を棒グラフまたは折れ線グラフで表示
     - チャートライブラリ: `recharts` を使用する
     - X 軸: 日付（日次なら日、週次なら週、月次なら月）
     - Y 軸（左）: リクエスト数
     - Y 軸（右）: 推定コスト（USD）
     - SNS 別に色分け
   - **テーブル**: プラットフォーム別の詳細数値（リクエスト数、成功数、失敗数、コスト）
   - **予算消費バー**: 各アクティブなポリシーの消費率をプログレスバーで表示
     - 80% 超: warning カラー
     - 100% 超: error カラー

2. **予算設定ページ** `settings/budget/page.tsx`
   - 既存ポリシー一覧テーブル
     - 各行: スコープ（ワークスペース/SNS/エンドポイント）、期間、上限額、超過時アクション、現在消費率
   - ポリシー作成フォーム（モーダル）
     - スコープ選択: ワークスペース全体 / プラットフォーム指定 / エンドポイント指定
     - 期間: 日次 / 週次 / 月次
     - 上限額（USD）
     - 超過時アクション: warn / require-approval / block
   - ポリシー編集・削除
   - 設定画面のタブに「予算」を追加する

3. **チャート描画**
   - `recharts` の ResponsiveContainer, BarChart, LineChart, XAxis, YAxis, Tooltip, Legend を使用
   - DaisyUI テーマの色をチャートに適用する
   - データが 0 件の場合は空状態メッセージを表示する
