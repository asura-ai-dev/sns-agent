# Task 4005 done_when

- `/usage` にアクセスすると使用量画面が表示される
- 日次/週次/月次の期間切り替えが動作し、表示が更新される
- プラットフォームフィルタで特定 SNS のみの使用量が表示される
- サマリカードに合計コスト、リクエスト数、成功率が表示される
- recharts でグラフ（棒グラフまたは折れ線グラフ）が描画され、SNS 別に色分けされている
- プラットフォーム別の詳細テーブルが表示される
- 予算消費バーが表示され、80% 超で warning カラー、100% 超で error カラーになる
- `/settings/budget` で予算ポリシーの一覧が表示される
- ポリシー作成フォームからスコープ、期間、上限額、超過時アクションを指定して作成できる
- デスクトップとモバイルの両方で表示が崩れない
